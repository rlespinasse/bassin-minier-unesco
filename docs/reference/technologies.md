# Technologies

## Frontend

| Technologie   | Version | Usage                              | Source    |
| ------------- | ------- | ---------------------------------- | --------- |
| [Leaflet](https://leafletjs.com/) | 1.9.4 | Carte interactive | CDN |
| [OpenStreetMap](https://www.openstreetmap.org/) | — | Fond de carte par défaut | Tuiles |
| HTML/CSS/JS   | —       | Interface utilisateur              | Vanilla   |

Trois fonds de carte sont disponibles : IGN Plan, CARTO Light et Esri Satellite.

## Pipeline de données

| Technologie   | Usage                                              |
| ------------- | -------------------------------------------------- |
| [GeoPandas](https://geopandas.org/) | Conversion de shapefiles, manipulation géométrique |
| [Python](https://www.python.org/)   | Scripts de téléchargement, conversion, enrichissement |

## Analytique

| Service       | Usage                                              |
| ------------- | -------------------------------------------------- |
| [GoatCounter](https://www.goatcounter.com/) | Suivi analytique respectueux de la vie privée |

Le suivi est désactivé automatiquement sur `localhost`.
