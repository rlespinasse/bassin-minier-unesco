# Raccourcis clavier

## Vue d'ensemble

La carte supporte des raccourcis clavier pour une navigation rapide. Appuyez sur `?` pour afficher l'aide contextuelle directement sur la carte.

## Raccourcis disponibles

| Touche    | Action                                                    |
| --------- | --------------------------------------------------------- |
| `E`       | Ajuster la vue à l'étendue du bassin minier               |
| `R`       | Réinitialiser la carte (recharge la page)                 |
| `F`       | Placer le curseur dans la barre de recherche              |
| `C`       | Ouvrir/fermer le panneau de couches                       |
| `P`       | Ouvrir/fermer le panneau de détail                        |
| `H`       | Naviguer vers l'élément précédent dans l'historique       |
| `J`       | Naviguer vers l'élément suivant dans l'historique         |
| `+`       | Zoomer                                                    |
| `-`       | Dézoomer                                                  |
| `?`       | Afficher/masquer l'aide des raccourcis                    |
| `Echap`   | Fermer l'aide, le panneau de couches ou le panneau de détail (dans cet ordre de priorité) |

## Notes

- Les raccourcis sont désactivés lorsque le curseur est dans un champ de saisie.
- La touche `Echap` ferme les éléments dans un ordre hiérarchique : aide > panneau de couches > panneau de détail.
