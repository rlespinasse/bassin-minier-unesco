# Bassin Minier du Nord-Pas de Calais - Patrimoine mondial UNESCO

Site web statique avec carte interactive présentant les données patrimoniales du Bassin minier du Nord-Pas de Calais, inscrit au patrimoine mondial de l'UNESCO en 2012.

## Démarrage rapide

```bash
mise install
just install
just convert
just serve
```

Ouvrir <http://localhost:8000> dans un navigateur.
